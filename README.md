# EdTech Student Completion Analysis 📊

## Overview
Analysis of 1,500 student records from an EdTech platform to identify 
course completion patterns and dropout factors using Python and Power BI.

## Key Insights
- 55% of students drop courses before completion
- AI/ML courses have highest dropout rate (63%)
- Desktop users complete more courses than mobile users
- Students scoring below 55 in quizzes are likely to drop out

## Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- Power BI (4-page interactive dashboard)
- SQL
- Excel

## Project Structure
- edtech_analysis.ipynb — Python analysis notebook
- EdTech_Analysis.pbix — Power BI dashboard
- data/ — Dataset files

## Dashboard Pages
1. Overview — KPIs and completion summary
2. Course Performance — Course level analysis
3. Student Demographics — Age, device, employment analysis
4. Business Insights — Recommendations
